Checar se fui exposto no dossiê de Anti-fascistas

No dia 3 de junho de 2020 ficamos sabendo que um dossiê com possíveis anti-fascistas foi exposto na internet.

Para checar se seu nome está no dossiê sem correr o risco de expor seus dados use esse serviço.

1. Gere um hash com o seu nome completo

Utilizamos sha512 para não expor seu nome. Você pode usar o Duck Duck Go para gerar um hash com o seu nome. Vale a pena gerar com variações de maiúscula e minúscula, com ou sem acento, etc.

2. Informe os 100 primeiros caracteres do seu hash

Por que hashes

Esse site não contém os dados presentes no dossiê, somente parte dos hashes dos nomes.

Isso garante que a publicação desse site não expõe as pessoas a um risco ainda maior.

O algoritmo usado foi o sha512./

Para verificar se os seus dados estão presentes, insira no campo os oito primeiros caracteres do hash do seu nome completo.

Essa é a única forma de garantir que seus dados não estão sendo compartilhados de forma imprópria.
